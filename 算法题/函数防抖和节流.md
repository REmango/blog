### 函数防抖和节流（debounce）

思路：这里采用 setTimeout 实现，会比较精简，当然也可以通过 2 次触发事件的时间点获取时间间隔实现。

```javascript
const debounce = (func, wait) => {
  let timer;
  return () => {
    clearTimeout(timer);
    timer = setTimeout(func, wait);
  };
};
```

```javascript
const throttle = (func, wait) => {
  let timer;

  return () => {
    if (timer) {
      return;
    }

    timer = setTimeout(() => {
      func();
      timer = null;
    }, wait);
  };
};
```
